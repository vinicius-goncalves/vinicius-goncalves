Linguagens

HTML

HTML é uma linguagem de marcação utilizada para construções de páginas web. É o/a HTML que define títulos, parágrafos e atualmente, no HTML5, toda a semântica por trás de um website.

CSS

CSS é uma linguagem para estilização, ou seja, a parte visual. Com ele é possível definir as cores, posições, criar animações e muito mais.

JavaScript

JavaScript é uma linguagem de programação que com ela podemos fazer toda a interatividade de um website, tudo isso de forma dinâmica. Hoje em dia, JavaScript também está presente fora dos navegadores, em milhões de aplicações.

SQL

SQL é uma linguagem utilizada na comunicação de banco de um dados relacional. Ela muito utilizada em vários RDBMS para a comunicação com um banco de dados.
